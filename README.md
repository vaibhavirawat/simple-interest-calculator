# Simple Interest Calculator

## Description
A simple calculator to calculate simple interest using principal amount, rate of interest, and time.

## Formula Used
Simple Interest = (P × R × T) / 100

## Features
- Calculate simple interest instantly
- Easy to use
- Beginner-friendly project

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository
2. Open index.html in your browser
